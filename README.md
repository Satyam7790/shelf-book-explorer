# Shelf 📚

### What is this?
**Shelf** is a simple website I built for my college project to help people find books without the clutter. It uses the **Google Books API** to pull real-time data, so you can search for a specific title or just browse through different categories like an encyclopedia.

###  What it can do:
* **Search Anything:** Type in a book name or an author and get results instantly.
* **Filter by Genre:** I set up specific categories so you can explore things like Science, Fiction, or Biography without typing a word.
* **Smart Loading:** Since there are millions of books, the app loads them in small batches so it doesn't crash your browser.
* **Book Details:** Click on a book to see the description, page count, and the cover art.
